# Customer Segmentation Analysis using K-Means Clustering

## Overview

This project performs customer segmentation using the K-Means Clustering algorithm. The objective is to group customers with similar purchasing behavior and demographic characteristics, enabling businesses to better understand customer groups and support data-driven decision making.

---

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Feature Engineering
- One-Hot Encoding
- Feature Scaling using StandardScaler
- K-Means Clustering
- Optimal Cluster Selection using:
  - Elbow Method
  - Silhouette Score
- PCA-based Cluster Visualization
- Cluster Analysis

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Workflow

1. Load and explore the dataset.
2. Perform data cleaning and preprocessing.
3. Handle missing values.
4. Encode categorical features.
5. Scale numerical features.
6. Select relevant features for clustering.
7. Determine the optimal number of clusters using the Elbow Method and Silhouette Score.
8. Train the K-Means model.
9. Visualize clusters using PCA.
10. Analyze customer segments.

---

## Results

The final model groups customers into meaningful segments based on their purchasing patterns and customer characteristics. Cluster quality was evaluated using the Silhouette Score, while the Elbow Method was used to determine an appropriate value of K.
